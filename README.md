**Лабораторные работы по низкоуровневому программированию 
Assembler x86-64**

**Лабораторная работа №1**
Нахождение максимального из трех чисел. Вычисление разницы между максимумом и каждым числом

**Лабораторная работа №2**
Ввод количества секунд (0-86400). Вычисление секунд с начала последнего часа

**Лабораторная работа №3**
Проверка возможности построения треугольника по трем сторонам. Определение типа треугольника (равносторонний, равнобедренный, разносторонний)

**Лабораторная работа №4**
Вычисление НОД двух чисел. Нахождение всех делителей НОД (положительных и отрицательных)

**Лабораторная работа №5**
Поиск простых чисел в заданном диапазоне методом квадратичного решета. Подсчет количества простых чисел. Вывод списка найденных простых чисел

**Лабораторная работа №6**
Вычисление среднего арифметического всех элементов массива, кроме заданного (индексом)

**Лабораторная работа №7**
Проверка симметричности квадратной матрицы относительно побочной диагонали

**Лабораторная работа №8**
Вычисление веса булевого вектора
